preparePlayer

  • preparePlayer(guild: string, textChannel: string, voiceChannel: string, options?: PlayerOptions): Promise<Player>
    • Defined in src/structures/Manager.ts:269

    Creates a new player and connects it to the voice channel. Also checks channel permissions. The player is not permanently saved or bound to the manager if it fails to connect or doesn't have sufficient permissions. If a player for the specified guild already exists, it is returned and no new player is created. If it is disconnected, it is automatically connected.

    Parameters

    • guild: string

      The player's guild.

    • textChannel: string

      The player's text channel.

    • voiceChannel: string

      The player's voice channel.

    • Optional options: PlayerOptions

      The player's options.

    Returns Promise<Player>

    The created player.

search

    • Defined in src/structures/Manager.ts:317

    Get search results based on a query. If the query is a link, it will attempt to get a track from the link. If not, it will return results from a search using the specified or default source.

    Parameters

    • query: string

      The query to search with.

    • Optional requester: string

      The user that requested the track. This value can be anything, and solely exists for your convenience.

    • Optional source: "yt" | "sc"

      The source to use if the query is not a link. Defaults to the manager's default source.
