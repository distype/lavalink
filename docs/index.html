@distype/lavalink



About

A Lavalink wrapper with native bindings to Distype. This library is currently tested and developed for Lavalink 3.4, using Java 13.

  • Docs

Installation

@distype/lavalink can be installed via npm.

npm install @distype/lavalink

Prerequisites
  • Node.js >=16.13.0
  • NPM >=8.1.0
  • Java ^13.0.0
  • Lavalink 3.4
